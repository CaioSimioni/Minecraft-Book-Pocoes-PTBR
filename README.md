# Minecraft Book Poções PTBR

![Capa do Livro](.public/capa.png)

Este projeto é um resumo das poções do jogo Minecraft, escrito por CaioSimioni.

O objetivo é apresentar, de forma simples e organizada, todas as poções do jogo, seus ingredientes, preparo e efeitos, seguindo o formato visual do caderno do Minecraft.

Ideal para consulta rápida, aprendizado e organização das receitas dentro das limitações do livro do jogo.
